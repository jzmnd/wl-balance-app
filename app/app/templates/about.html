    About the Author

    Jeremy Smith

    I am a multidisciplinary scientist with experience analyzing and modeling complex electronic systems. Before becoming interested in the field of data science I studied materials science and physics and worked on challenges related to printed and flexible electronics. I am currently aiming to transition some of these skills into solving cutting edge problems in data science. In particular, my interests are in machine learning, neural networks, and NLP, and how to apply these techniques in as many diverse areas as I can!

    This project is part of my capstone for The Data Incubator program (Fall 2017).

    This project was built using...

    • Python / numpy / scipy / pandas - the usual suspects
    • Scikit learn - for regression and classification models
    • postgreSQL - for managing the ATUS data
    • Flask - for building the web app
    • Bootstrap - layout for the app
    • Docker - containers for deploying the models and predictors online
    • Bokeh - for basic plotting with Flask
    • d3 - for interactive bubble plot visualizations

    All data is from the American Time Use Survey (ATUS) that is published by the US Bureau for Labor Statistics. Daily activity times for respondents between 2003 and 2015.
